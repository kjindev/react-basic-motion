# react-basic-motion

The Simple React CSS Component Library

## Installing

    npm install react-basic-motion

## Features

- React CSS Component
- TypeScript Support

## Docs

[react-basic-motion](https://react-basic-motion-docs.vercel.app/)

## Simple Example

```typescript
<Bounce
    name="x"
    range={ 10 }
    duration={ 1.5 }
    timing="ease-in-out"
    count="infinite"
>
    <div>Bounce!<div>
</Bounce>
```

Learn more it in docs!
